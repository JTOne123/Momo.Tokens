## What the momo?

Various configurations suitable for depency injection used in [goto10](http://www.goto10.cz) libraries.

## List of configurations

- [ShushuConfiguration](https://github.com/goto10hq/Shushu)
  - Name
  - ServiceApiKey
  - SearchApiKey
  - Index
- StorageConfiguration
  - ConnectionString
  - Container
